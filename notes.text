When making a commit:
- git add .
- git commit -m 'message'
- git pull origin master -> when you are working with a team (it will pull changes form other people)
- git push origin main

Getting rid of a change:
  - if already used git add <file_name> to UNSTAGE
    git reset HEAD <file_name>
  - if already UNSTAGED
    * git checkout -- <file_name>
